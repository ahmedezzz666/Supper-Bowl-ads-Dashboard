# Super Bowl Ads Dashboard

Interactive **Super Bowl Ads Dashboard** built in Power BI to analyze advertising performance and cost across brands, industries, and years.

## Project Overview

This dashboard explores Super Bowl commercials using multiple perspectives:
- Performance by year, brand, and industry.
- Ads cost by year, brand and industry

The goal is to quickly spot which ads performed best and identify patterns in successful Super Bowl campaigns.

## Data & Tools

- **Tool:** Power BI Desktop (DAX + Power Query).
- **Dashboard file:** `SuperBowel_Ezz.pbix`.
- **Data source:** Super Bowl ads dataset.  

## How to Use

1. Download the file `super-bowl-ads-dashboard.pbix` from this repository.
2. Open it in **Power BI Desktop**.
3. Use the buttons (ad cost, Brand, #ads, overview, Details, year, etc.) to explore different ad segments.
4. Hover over visuals for detailed tooltips and insights.
